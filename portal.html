1.1 — What is Data & Data Analysis?
📁 Module 1: Foundations ⏱ 12 min ✓ Free preview
Start here. This lesson defines what data is, what data analysis means in the real world, and why it matters in everyday life and business. By the end you'll understand the difference between data, information, and insight — and why companies pay analysts to turn one into the other.
1
Knowledge Check
Answer all questions · Pass mark: 80%
1. Which of the following best describes the role of a data analyst?
Writing code to build mobile apps
Collecting, cleaning and interpreting data to help businesses make decisions
Designing user interfaces and experiences
Managing servers and cloud infrastructure
2. What is the difference between data and information?
They mean exactly the same thing
Data is raw facts and figures; information is data that has been processed into something meaningful
Information is always numerical; data is always text
Data is only used by scientists; information is for businesses
3. Which of these is an example of quantitative data?
A customer's name
A product's colour
Monthly sales figures
A customer review comment
2
Reflection Prompt
Private — only you can see this
Think about your own daily life or your current/previous job. Where do you encounter data without realising it? How could data analysis help you make better decisions in that area?
